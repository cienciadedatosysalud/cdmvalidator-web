### Project Description
**Common Data Model Validator - Web Version** is a specialized web application designed to work in tandem with the [Common Data Model Builder - Web Version](https://cienciadedatosysalud.github.io/cdmb-web/?lng=en).

This tool empowers users to define complex **validation rules** using standard SQL syntax, leveraging the power and flexibility of **DuckDB** (see [DuckDB Documentation](https://duckdb.org/docs/sql/introduction)). Additionally, it provides advanced capabilities to modify and refine synthetic datasets through SQL `UPDATE` commands, allowing for the simulation of specific data scenarios and edge cases.

### Key Features
* **SQL-Based Validation:** Create precise and complex validation logic using DuckDB SQL syntax.
* **Advanced Data Manipulation:** Modify synthetic data directly using `UPDATE` statements to test specific conditions or simulate dirty data.
* **Seamless Integration:** Designed to produce outputs that are directly compatible with the Common Data Model Builder.

### Outputs
The application generates two main types of outputs:
1.  **Validation Rule Set:** A structured set of rules ready to be imported directly into the **Common Data Model Builder - Web Version**.
2.  **Modified Synthetic Dataset:** A dataset containing the synthetic records modified according to the user's SQL update commands.
